# 评委快速启动卡片

## CivicMate 办事不跑空

广东省县级政务材料预审与陪办助手 Demo。

---

## 快速启动

```bash
npm install
npm run dev
```

打开浏览器访问：**http://localhost:5173**

## 推荐演示路径（约 2 分钟）

1. **点击一键演示场景**：首页选择"老年补贴申请 / 居住证办理 / 医保报销"任一场景，系统自动填充并跳到第 5 步。
2. **点击"生成预审结果"**：触发本地规则预审，结果页立即展示。
3. **查看缺失材料**：红色块标注缺失项，橙色块标注待确认项。
4. **查看老人友好清单**：蓝色块列出简化版材料清单，适合老年人阅读。
5. **点击"复制清单"**：纯文本清单复制到剪贴板，页面显示绿色"已复制到剪贴板"。
6. **点击"打印页面"**：调起浏览器打印，步骤/导航/上传区已隐藏。

## 未配置 API Key 时的表现

- AI 状态徽标显示"本地规则仍可演示"。
- 预审结果使用本地规则，`aiStatus=skipped`。
- 材料上传识别仅保留用户手动标注。
- **演示主流程完全不受影响**，评委无需准备 Key。

## 配置 API Key 后可启用的增强能力

在 `.env` 中配置后重启服务：

| Key | 增强能力 |
|-----|----------|
| `DEEPSEEK_API_KEY` | 真实 AI 预审增强（政务规则解释/材料审查） |
| `ARK_API_KEY` | 图片材料视觉识别（Ark 豆包视觉模型） |

配置后 AI 状态徽标显示"预审增强可用 / 图片识别可用"。

## 重要提示

- **结果不代表官方审核通过**，仅供材料准备参考。
- 正式办理请以广东当地办事机关要求为准。
- 上传文件不保存到磁盘，仅内存处理。
